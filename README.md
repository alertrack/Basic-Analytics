# Basic-Analytics
Desafio de Lógica de Programação e Analise de dados básico - utilizando a linguagem Python

**Tarefas**:

1. Calcular o número total de transações no conjunto de dados.
2. Calcular a média e o desvio padrão das transações no conjunto de dados.
3. Identificar os 10 clientes com a maior quantidade de transações e identificar suas respectivas médias transacionais.
4. Criar uma nova coluna no DataFrame com a data mais recente em que cada cliente realizou uma transação.
5. Criar um gráfico de dispersão com a recência e a quantidade média de transações por cliente.
6. Normalização de datas para o formato yyyy-mm-dd

**Importante**: Para a manipulação da recência, considere a data mais recente do dataframe como a data base para o cálculo.

**Roteiro**:

* Consumir o arquivo basic-analitycs.csv.
* Carregar o arquivo csv como um DataFrame.
* Fazer uma breve análise exploratória dos dados para entender o que está contido no arquivo.
* Criar insights pessoais a partir das análises realizadas e compartilhá-los em um documento separado no formato Markdown
